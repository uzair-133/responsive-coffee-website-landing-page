# ☕ Coffee House - Premium Landing Page

A high-performance, fully responsive landing page designed for a boutique coffee brand. This project emphasizes clean UI/UX, fast loading speeds, and a modern "Old Money" aesthetic.

## Live Preview
Check out the live site here: [Luminous Coffee House](https://luminous-sfogliatella-a04d53.netlify.app/)

## Key Features
- **Modern UI:** Minimalist and elegant design tailored for high-end brands.
- **Responsive Layout:** Works perfectly on Mobile, Tablets, and Desktops.
- **Interactive Menu:** Categorized food and beverage sections.
- **Social Proof:** Testimonials section to build customer trust.
- **SEO Friendly:** Optimized semantic HTML structure.

## Tech Stack
- **Languages:** HTML5, CSS3
- **Tools:** VS Code, Git/GitHub
- **Deployment:** Netlify
